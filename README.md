
## INSTALL DEPENDENCIES AND CREATE MIGRATIONS 

This application use Laravel and SQLITE DATABASE

```bash
-- npm install

-- composer update

-- php artisan migrate

-- php artisan serve

``` 


## START FRONT END 

```bash

-- npm run dev
```
